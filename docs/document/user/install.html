# 安装

# 下载程序

  • 在官网下载 📦 BookMate 压缩包（如果从 Github 下载速度较慢，可以切换 Gitee 或 坚果云 下载源）
  • 在 MongoDB 官网open in new window下载安装包

# 安装程序

压缩包解压后有两个文件夹 📁 BookMate-Client 和 📁 BookMate-Server ，分别与客户端和服务端相关

  • 📁 BookMate-Client
    • 可以运行程序 💾 bookmate-client Setup 进行安装
    • 也可以不安装，直接使用未打包的文件夹 📁 bookmate-client-unpacked
  • 📁 BookMate-Client
    • 服务端不需要安装，程序 💾 bookmate-server 就是服务端程序
    • 文件夹 📁 public 用于存放用户上传的图片、视频等数据 ⚠️ 文件夹 📁 public 需要和程序 💾 bookmate-server 存放在同一级目录下
  • 💾 MongoDB 是一款开源的数据库，其的安装请参考官方文档open in new window

# 运行程序 🎥open in new window 📺open in new window

由于 BookMate 采用前后端分离的设计，因此需要依此启动 MongoDB、BookMate-Server、BookMate-Client 程序。

  • MongoDB 程序启动方法参考官方文档open in new window 💡 安装 MongoDB 时如果选择 Run service as Network Service user，则 MongoDB 会在开机时自动运行，不需要每次手动开启
  • BookMate-Server：运行 💾 bookmate-server 程序启动服务端
  • BookMate-Client：运行 💾 bookmate-client 程序启动客户端

💡 如果客户端界面无法加载数据，可以按 F5 快捷键刷新页面；如果刷新后仍未成功加载数据，在开启数据库 MongoDB 的前提下，可以尝试重启 💾 bookmate-server 程序，再按 F5 快捷键刷新客户端

# 分享

# 书摘分享 🎥open in new window 📺open in new window

可以将单篇/多篇书摘分享为图片、Markdown 文件、JSON 文件

  • 单篇书摘分享：点击书摘卡片右下加的分享图标按钮，选择希望分享的格式
  • 多篇书摘分享：结合 Pin 功能open in new window，可以实现多篇书摘分享

💡 可以点击 「书摘」 栏目右上角的汉堡菜单（三等号）按钮，在弹出框中的 「分享书摘设置」 条目下选择需要设置的分享格式，在弹出框中依照提示设置分享参数

# 概述分享

可以将单篇概述分享为 Markdown 文件。点击概述卡片右下加的下载图标按钮。
